## Tree keymaps

| Keymap       | desc                                         |
| ------------ | -------------------------------------------- |
| \<leader\>ee | Open file tree                               |
| \<leader\>ef | Open file tree Highlighting the current file |

| Keymap       | desc                       |
| ------------ | -------------------------- |

| \<leader\>pr | Find recently opened files |
| \<leader\>ps | Find words                 |
| \<leader\>pt | Find "TODO"s               |
